# オーナー様向け 操作マニュアル

このシステムでできること:

1. お客様からのLINEの質問に、24時間自動でお返事します
2. 答えられない質問は、オーナー様のLINEに「【要対応】」と通知が届きます
3. 質問と答え・メニューと料金は、スマホからいつでも変更できます
4. 臨時休業などのお知らせを、友だち全員に一斉送信できます

すべてスマホだけで操作できます。パソコンは不要です。

## 管理画面の入り方

1. スマホのブラウザで管理画面のURLを開く（ホーム画面に追加しておくと便利です）
   - URL: https://salon-line-bot-five.vercel.app/admin
2. ユーザー名とパスワードを聞かれたら入力する
   - ユーザー名: 何でもOK（例: owner）
   - パスワード: 別途お渡ししたもの（他の人に教えないでください）

＜画面キャプチャ: 管理画面トップ（3つのメニューが並んだ画面）＞

## よくある質問を変えたいとき

例）駐車場の案内を変えたい、新しい質問を足したい

1. 管理画面の「よくある質問」を開く
2. 変えたい質問をタップ → 内容を直して「保存する」
3. 新しく足すときは「＋ 新しい質問を追加する」をタップ

＜画面キャプチャ: よくある質問の編集画面（質問をタップして開いた状態）＞

保存した瞬間から、botのお返事に反映されます。

## 料金を改定したいとき

1. 管理画面の「メニュー」を開く
2. 変えたいメニューをタップ → 料金を直して「保存する」

これだけで、お客様への料金のご案内が新しくなります。

## お知らせを一斉送信したいとき

例）臨時休業、営業時間の変更、キャンペーン

1. 管理画面の「お知らせ」を開く
2. 内容を入力する
3. **まず「自分にテスト送信」を押す** → 自分のLINEに届くので、見え方を確認
4. 問題なければ「全員に配信する」を押す

＜画面キャプチャ: お知らせ配信画面（テスト送信と全員配信の2つのボタン）＞

注意:
- **配信は取り消せません。** 必ずテスト送信で確認してからにしてください
- LINEの無料プランで送れるのは月200通までです。友だち300人に配信すると1回で300通になるため、配信を使う月はLINEの有料プラン（コミュニケーションプラン等）への切り替えが必要です

## 「【要対応】」の通知が届いたら

botが答えられなかった質問です。以下の対応をお願いします。

1. LINE公式アカウントの管理アプリ（LINE Official Account）を開き、そのお客様に直接返信する
   - お客様には「オーナーに確認して折り返しご連絡します」と自動でお伝えしてあります
2. 同じ質問が何度も来るようなら、管理画面の「よくある質問」に答えを追加する
   → 次からはbotが自動で答えます

## 毎月の費用

- AIの利用料: 月数百円〜（質問の件数によります。1日10〜15件なら数百円程度）
- サーバー・データベース: 無料枠内
- LINE: 自動応答だけなら無料。一斉配信を使う月のみ有料プラン

## 困ったときは

- botが返事をしない / 管理画面が開かない → 開発担当までご連絡ください
- パスワードを忘れた → 開発担当で再設定できます
