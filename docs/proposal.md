# LINE bot + AI自動応答 ご提案書

## 1. 現状の課題

- 1日10〜15件のLINE問い合わせに、オーナー様がすべて手動で対応
- 施術中や営業時間外も返信に追われ、本来のお仕事に集中できない
- 質問の多く（営業時間・料金・駐車場など）は毎回同じ内容の繰り返し

## 2. ご提案する解決策

**やること:**
- よくある質問への自動応答（AIが24時間、自然な日本語で回答します）
- botが答えられない質問はオーナー様のLINEにお知らせ（お客様を待たせません）
- FAQやメニュー料金をスマホから更新できる管理画面（営業時間変更や料金改定の際も、業者へ依頼せずご自身で数分で更新できます）
- 臨時休業などのお知らせを友だち全員（約300人）へ一斉配信

**やらないこと（今回の範囲外）:**
- 予約の自動受付（現在どおり電話・LINEでの手動対応。Phase 2でご相談可能です）
- 顧客カルテ・決済機能

## 3. 導入後に見込める効果

- 1日10〜15件の問い合わせのうち、**約8割を自動応答**
- オーナー様の対応時間: **1日約30分（1件2〜3分と想定した場合）→ 5分以下**（月に換算すると約12時間の削減）
- 営業時間外の問い合わせにも即回答 → 機会損失を防止

※上記の効果は、よくある質問が問い合わせの大半を占めることを前提とした想定値です。

## 4. 仕組みと使う技術

- LINE公式アカウント（現在お使いのものをそのまま活用します）
- AI: Claude（自然な日本語で応答するAIです）
- サーバー・データベース: Vercel／Supabase（どちらも無料枠で運用できます）

## 5. 導入後の流れ

1. FAQ内容のヒアリング（よくある質問と回答を一緒に整理します）
2. bot構築・FAQ登録
3. テスト運用（オーナー様に実際のLINEで動作をご確認いただきます）
